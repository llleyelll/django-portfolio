# Portfolio

## Prerequisite
- Python 3.8+

## Run locally

Clone and change path to the project directory
```
$ git clone https://github.com/llleyelll/django-portfolio.git && cd django-portfolio
```
Install dependencies
```
$ pip3 install -r requirements.txt
```
Run database migrations by the following commands
```
$ python3 manage.py makemigrations
$ python3 manage.py migrate
```
Run server and server will up at http://127.0.0.1:8000/
```
$ python3 manage.py runserver
```
